Recipe Book

This project includes working with integrating python backend development with frontend HTML, CSS & Javascript. It also introduces the concept of jijna templating.


The tools used for this project are Python, MongoDB, Jinja templating along with HTML,CSS & Javascript.

The framework used was Flask which is quite popular among python developers to create small scale projects. The backend involves adding CRUD functionalities (Create, Read, Update & Delete) while the frontend involves rendering the content that we get back from the database and displaying it to the user. It also provides user with the ability to add information(recipes) to the database, edit & delete if required.
